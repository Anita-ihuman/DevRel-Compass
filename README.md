# DevRel Compass 🧭

> Open-source career development platform for Developer Relations practitioners — skills assessment, career roadmap, and resources.

DevRel Compass helps Developer Relations professionals understand where they
stand and where to grow. Upload your resume for an AI-powered assessment across
the core DevRel skill dimensions — benchmarked against global standards from
companies like CNCF, Google, AWS, Stripe, Twilio, and HashiCorp — then follow a
structured career roadmap to level up.

## Who is this for?

DevRel Compass is for anyone who is:

- 📌 Trying to build a career in DevRel
- 📌 Applying for a job as a developer advocate or DevRel professional
- 📌 Researching how DevRel roles are evolving and how to keep up with the trend
- 📌 Looking to contribute to an open source project

## Features

- **Skills Analyzer** — Upload a resume (PDF/DOCX) and get a precision score
  across each DevRel skill dimension, including strengths, gaps, and a career
  level estimate. Powered by Claude.
- **Job Fit** — See how your profile maps against DevRel role expectations.
- **Career Roadmap** — An interactive, phase-by-phase roadmap of the skills and
  topics that matter at each stage of a DevRel career.
- **Events** — A curated list of community events, webinars, and conferences.
